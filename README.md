*Se ha incluido un archivo .env.example con la estructura necesaria.

*Renombrar el archivo ".env.eaxmple" a ".env" y agregar el "MONGO_URI" que se encuentra en el correo.

*En dicho archivo ".env" colocar el "JWT_SECRET" que usted considere (o se puede dejar la que se muestra de ejemplo).

*Para ejecutar el proyecto, abra dos terminales: una en la carpeta del backend y otra en la carpeta frontend.

*Ejecutar

*git clone <vr-in-situ-prueba-frontend>

*cd backend-vris-final

*npm install

*npm run dev

*He configurado el acceso a la base de datos de MongoDB Atlas para permitir conexiones desde 
cualquier IP (0.0.0.0/0).

*El frontend está configurado para conectarse al backend en http://localhost:8001. Por favor, asegúrese de que el servidor backend esté corriendo en dicho puerto.
